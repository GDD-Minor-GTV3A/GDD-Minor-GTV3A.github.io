InputHandler

class in Core

/

Implemented in: Core.InputSystem

Description

ScriptableObject holding callbacks for the player input.

Referencing this will grant access to every input event that has been defined. Those events can easily be modified when needed

Public Methods

Method Description
MovementEvent Gameplay input. Event that listen to whenever the player moves. Returns a Vector2.
LookEvent Gameplay input. Event that listen to whenever the mouse/right stick moves. Returns a Vector2.
MainUsageEvent Gameplay input. Event that listen to the press of the main usage button.
MainUsageCanceledEvent Gameplay input. Event that listen to when the main usage button is let go.
SecondaryUsageEvent Gameplay input. Event that listen to the press of the main usage button.
SecondaryUsageCanceledEvent Gameplay input. Event that listen to when the main usage button is let go.
InteractEvent Gameplay input. Event that listen to the press of the interact button.
SprintEvent Gameplay input. Event that listen to the press and release of the sprint button. Returns a boolean.
ReloadEvent Gameplay input. Event that listen to the press of the interact button.
SlotsEvent Gameplay input. Event that listen to a slot button is pressed. Returns an int depending on the pressed slot.
SlotsScrollEvent Gameplay input. Event that listen to the scroll wheel to change slots. Returns a Vector2.
BarkEvent Gameplay input. Event that listen to the press of the bark button.
NavigateEvent UI input. Event that listen to the press of the navigation buttons. Returns a Vector2.
