# PersonalityType

- **Namespace:** AI
- **Implemented in:** `Core.AI.Sheep.Config`

Enum describing the available personality profiles for sheep. Personality affects how sheep react to stimuli. Configure it in the `SheepArchetype` ScriptableObject.

## Members

| Member | Description |
| --- | --- |
| `Normal` | Baseline behaviour. |
| `Lazy` | Slower, less reactive behaviour. |
| `Energetic` | More active behaviour. |
| `Nervous` | Skittish, prone to reacting quickly. |
| `Stubborn` | Resistant to change, slower to react. |
