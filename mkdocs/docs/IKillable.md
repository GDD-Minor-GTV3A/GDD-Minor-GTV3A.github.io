# IKillable

- **Namespace:** Gameplay.HealthSystem

Interface for components that can die and broadcast corresponding events.

## Members

| Member | Description |
| --- | --- |
| `DeathEvent` | UnityEvent raised when the implementer dies. |
| `Die()` | Handles the death logic. Call to kill the implementer. |
