# CameraType

- **Namespace:** Gameplay
- **Implemented in:** `Gameplay.CameraSettings`

Enum for the mode of the camera. Configure the value in the `CameraConfig` ScriptableObject to switch camera behaviour at runtime.

## Members

| Member | Description |
| --- | --- |
| `Orthographic` | Sets the camera to orthographic projection. |
| `Perspective` | Sets the camera to perspective projection. |
