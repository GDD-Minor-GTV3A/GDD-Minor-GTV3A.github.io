# IHiddenObject

- **Namespace:** Core.Shared

Interface for systems that work with objects that can toggle their visibility in the fog of war.

## Members

| Member | Description |
| --- | --- |
| `DynamicallyAddHiddenObject()` | Adds the object to the fog-of-war system at runtime. |
| `DynamicallyRemoveHiddenObject()` | Removes the object from the fog-of-war system at runtime. |
| `GetPosition()` | Returns the position of the rendered object. |
| `SetVisible(bool visible)` | Controls whether the object is visible (`true`) or hidden (`false`). |
