# ScareType

- **Namespace:** Gameplay.Dog

Enum describing different scare sources. Passed as a parameter to [`IScareable`](IScareable.md).

## Members

| Member | Description |
| --- | --- |
| `DogBark` | Scare caused by a dog's bark. |
| `Growl` | Scare caused by a growl. |
| `Explosion` | Scare caused by an explosion. |
| `Gunshot` | Scare caused by a gunshot. |
| `Thunder` | Scare caused by thunder. |
| `Other` | Scare caused by another source. |
