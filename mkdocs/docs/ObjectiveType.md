# ObjectiveType

- **Namespace:** Gameplay

Enum representing the objective types used when defining quests.

## Members

| Member | Description |
| --- | --- |
| `CollectItem` | Quest requires collecting specific items. |
| `DefeatEnemy` | Quest requires defeating one or more enemies. |
| `ReachLocation` | Quest requires reaching a location. |
| `TalkNPC` | Quest requires talking to an NPC. |
| `Custom` | Quest uses custom completion logic. |
