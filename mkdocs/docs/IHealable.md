# IHealable

- **Namespace:** Gameplay.HealthSystem

Interface for components that can be healed.

## Members

| Member | Description |
| --- | --- |
| `HealEvent` | UnityEvent invoked when healing is applied. Use this to drive secondary effects. |
| `Heal(float amount)` | Applies healing to the implementer. Call to restore health by the given amount. |
