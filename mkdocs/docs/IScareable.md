# IScareable

- **Namespace:** Gameplay.Dog

Interface for objects that can be scared by the dog or other sources.

## Members

| Member | Description |
| --- | --- |
| `OnScared(Vector3 sourcePosition, float intensity, ScareType scareType)` | Handles how the implementer reacts to a scare, receiving the source position, intensity, and [`ScareType`](ScareType.md). |
