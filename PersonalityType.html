PersonalityType

class in AI

/

Implemented in: Core.AI.Sheep.Config

Description

Enum for available personality types for the sheep.

Changing personality will sligtly change how the sheep will react. Sheep personality type can be set in the SheepArchetype scriptableObject.

Public Methods

Method Description
Normal Set the type of personality to a normal personality.
Lazy Set the type of personality to a lazy personality.
Energetic Set the type of personality to a energetic personality.
Nervous Set the type of personality to a nervous personality.
Stubborn Set the type of personality to a stubborn personality.
