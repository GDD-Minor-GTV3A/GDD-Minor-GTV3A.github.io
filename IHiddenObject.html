IHiddenObject

interface in Core.Shared

Description

Interface for class capable of using objects that can be hidden.


Public Methods

Method Description
DynamicallyAddHiddenObject Dynamically adds hidden object to fog of war.
DynamicallyRemoveHiddenObject Dynamically removes hidden object from fog of war.
GetPosition Returns the position of the rendered object
SetVisible Sets visibility of this object. Takes a bool as parameter, where true = visible and false = not visible.
