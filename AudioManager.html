AudioManager

Implemented in: Core

Description

Manage audio for the entire game.

To implement the AudioManager just assign the OutputSource in the AudioSource of the object that is playing the audio to the corresponding source.

Ambience is for ambience sound, Music is for external music (doesn't scale with audio settings), Player, Dog and Enemy are sound effect channels for the respective player, dog and enemies.
Playing UI audio is handled inside the manager itself with the call of the function PlayUIAudio by passing a clip.

The AudioManager also include a simple script for playing audio without using the library UnityEngine.Audio.

Public Methods

Method Description
PlayAudio This method requires a clip and the audio source and like the UnityEngine.Audio.Play() is responsible to play the clip within that source. Optionally a bool can be passed with the previous params to set whenever the audio should loop.
PlayAudioAtPoint Like PlayAudio this method requires a clip and the optional bool for looping, but instead of the audio source it requires the position in Vector3 to where to instantiate the sound. (doesn't scale with settings).
StopLoopedAudio This method is responsible to stop a looped audio and requires the specific audio source that audio is on.
