Implemented in: Gameplay.Player

Description

Class responsable to read and broadcast player inputs.

Holds every input action for the player and returns it with public getters for each action. This script is responsable to initialize all player input actions and manage the cursor position in the game world.

Public Methods

Method Description
Move Returns a Vector2 value for the movement input of the player.
Look Current position of the cursor in the world. Returns an Observable.
Run Getter for the run button. Returns true when sprint button is hold, otherwhise false.
