IPlayerTool

interface in Core.Shared

Description

Interface for every tool that can be used by character.


Public Methods

Method Description
HideTool Use this to hide the character tool.
MainUsageFinished Called when left mouse button is released.
MainUsageStarted Called when the left mouse button is pressed. Takes an Observable as parameter.
Reload Logic of tool reload.
SecondaryUsageFinished Called when the right mouse button is released.
SecondaryUsageStarted Called when the right mouse button is pressed. Takes an Observable as parameter.
ShowTool Use this to show the character tool.
TryBark Use this to make your dog bark.
