IState

interface in Core.Shared.StateMachine

Description

Interface for states which are used in state managers.


Public Methods

Method Description
OnStart This method is called when one enter that state.
OnStop This method is called when one exit that state.
OnUpdate This method is called after the OnStart method and is called for each frame of the state or untile the OnStop method is called.
